# Sistema web PHP + MySQL

Sistema web desenvolvido utilizando PHP e MySQL para manipulação e visualização de dados em banco de dados remoto.

Este projeto foi utilizado em diferentes atividades do curso técnico no SENAI, sendo adaptado ao longo do tempo para atender novos requisitos, o que resultou em uma estrutura flexível e reutilizável.

## Objetivo

Simular um sistema utilizado por uma empresa fictícia, permitindo que usuários possam:

- visualizar dados armazenados em banco de dados
- cadastrar novas informações
- editar registros existentes
- excluir dados
- acessar informações organizadas em dashboard

O sistema foi desenvolvido com foco no aprendizado de integração entre frontend e backend utilizando PHP conectado a um banco MySQL.

## Tecnologias utilizadas

- PHP
- MySQL
- HTML
- CSS
- XAMPP (Apache + MySQL)
- phpMyAdmin
- FreeDB (banco de dados remoto)

## Funcionalidades

- CRUD completo (Create, Read, Update, Delete)
- conexão com banco de dados remoto
- organização de dados em tabelas
- dashboard com visualização de informações
- reutilização do mesmo sistema em múltiplas atividades
- estrutura preparada para integração com aplicação mobile

## Estrutura do projeto

O projeto segue a estrutura comum de aplicações PHP:

- arquivos de conexão com banco de dados
- páginas responsáveis por cada operação (cadastro, listagem, edição)
- organização modular para facilitar manutenção
- integração com banco MySQL

## Como executar o projeto

1. Instalar o XAMPP
2. Ativar Apache e MySQL
3. Importar o banco de dados pelo phpMyAdmin
4. Colocar a pasta do projeto dentro de: xampp/htdocs/
5. Acessar no navegador: http://localhost/phpMySQL

## Observações

Este projeto foi desenvolvido com fins educacionais, com foco no aprendizado de:

- integração entre PHP e banco de dados MySQL
- manipulação de dados via aplicação web
- estruturação de sistemas web simples
- reutilização de código em diferentes cenários
